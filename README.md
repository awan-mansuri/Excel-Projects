# 📊 Budget Dashboard — Excel Project 1

## Tools Used
Excel — SUMIF, SWITCH, Conditional Formatting, Charts

## Dataset
Personal Finance Dataset — Kaggle

## Questions Answered
- Which category overspent the most?
- What % of budget used per category?
- Total budget vs actual spending?

## Key Finding
Overspent by $83 total.
Restaurants, Shopping and Fast Food
were the biggest budget leaks.
