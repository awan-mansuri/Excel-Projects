## 📁 Project 1 — Personal Budget Dashboard

### 📌 Objective
Analyze personal budget vs actual spending to identify 
overspending categories and savings opportunities.

### 📂 Dataset
Personal Finance Dataset — Kaggle

### ❓ Questions Answered
- Which category overspent the most?
- What percentage of budget was used per category?
- Total budget vs actual spending comparison?
- Which categories are on track vs overspent?

### 💡 Key Findings
- Total spending exceeded budget by $83
- Restaurants overspent by 40%
- Shopping overspent by 45%
- Fast Food overspent by 87%
- Mortgage takes 47% of total budget

### 🛠️ Skills Used
- SUMIF, COUNTIF, SWITCH formula
- Conditional Formatting
- Clustered Bar Chart
- Pie Chart
- Budget Utilization Bar Chart
- KPI Boxes
- Dashboard Design

## 📁 Project 2 — Student Performance Dashboard

### 📌 Objective
Analyze student exam performance across Math, Reading 
and Writing based on gender, lunch type, test 
preparation and parental education level.

### 📂 Dataset
Students Performance in Exams — Kaggle
1000 students data

### ❓ Questions Answered
- Which gender performs better per subject?
- Does test preparation improve scores?
- Does lunch type affect performance?
- Does parental education impact scores?

### 💡 Key Findings
- Females score higher in Reading and Writing
- Males score higher in Math
- Test prep improves scores by 5-10 points
- Standard lunch students score 10+ points higher
- Master degree parents produce highest scoring kids
- High school educated parents = lowest scoring kids

### 🛠️ Skills Used
- AVERAGEIF formulas
- Power Query data cleaning
- Pivot Tables
- Interactive Slicers
- Clustered Bar Charts
- KPI Boxes
- Multi-chart Dashboard Design
