 📁 Project 1 — Personal Budget Dashboard
 🟢 Level: Beginner

 📌 Objective
Analyze personal budget vs actual spending to identify
overspending categories and savings opportunities.

 📂 Dataset
Personal Finance Dataset — Kaggle
19 expense categories

 ❓ Questions Answered
- Which category overspent the most?
- What percentage of budget used per category?
- Total budget vs actual spending comparison?
- Which categories are on track vs overspent?

 💡 Key Findings
- Total spending exceeded budget by $83
- Restaurants overspent by 40%
- Shopping overspent by 45%
- Fast Food overspent by 87%
- Mortgage takes 47% of total budget

 🛠️ Skills Used
- SUMIF, COUNTIF, SWITCH formula
- Conditional Formatting
- Clustered Bar Chart

- Budget Utilization Bar Chart

 📁 Project 2 — Student Performance Dashboard
 🟡 Level: Intermediate

 📌 Objective
Analyze student exam performance across Math, Reading
and Writing based on gender, lunch type, test
preparation and parental education level.

 📂 Dataset
Students Performance in Exams — Kaggle
1,000 students data

 ❓ Questions Answered
- Which gender performs better per subject?
- Does test preparation improve scores?
- Does lunch type affect performance?
- Does parental education impact scores?

 💡 Key Findings
- Females score higher in Reading and Writing
- Males score higher in Math
- Test prep improves scores by 5-10 points
- Standard lunch students score 10+ points higher
- Master degree parents produce highest scoring kids
- High school educated parents = lowest scoring kids

 🛠️ Skills Used
- AVERAGEIF formulas
- Power Query data cleaning
- Pivot Tables
- Interactive Slicers
- Clustered Bar Charts
- KPI Boxes
- Multi-chart Dashboard Design

 📁 Project 3 — Superstore Sales Dashboard
 🔴 Level: Advanced

 📌 Objective
Analyze 4 years of Superstore sales data to identify
revenue trends, profit drivers, regional performance
and customer segment insights.

 📂 Dataset
Superstore Sales Dataset — Kaggle
9,994 orders | 2014 to 2017 | 21 columns

### ❓ Questions Answered
- Which region generates highest sales and profit?
- Which category is most profitable?
- Which months have peak sales every year?
- Which customer segment drives most revenue?
- Which sub-categories are losing money?
- What is overall profit margin?

 💡 Key Findings
- West region has highest total sales
- Technology has highest profit margin
- Consumer segment drives 51% of revenue
- November and December are peak sales months
- Tables and Bookcases generate negative profit
- Copiers have highest profit per sub-category
- Overall profit margin is 12.47%
- Furniture has very low profit despite high sales

 🛠️ Skills Used
- Power Query (data cleaning, date extraction)
- Pivot Tables (5 pivot tables)
- PivotCharts (6 charts)
- Interactive Slicers (5 slicers)
- KPI Box Design
- SUMIF, COUNTA, SUM formulas
- Conditional Formatting
- Professional Dashboard Design
